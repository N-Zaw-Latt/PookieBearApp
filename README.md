# Pookie Bear Project
Requirements for the project
- Node Version 22.19 --> install LTS version from official site
- Git


Clone the Repository
'''
git clone https://github.com/leonyi4/PookieBearApp.git
cd your-repo-name
'''

Install Dependencies
`npm install`

Run Dev Server
`npm run dev`

Go to 
`http://localhost:5173`
